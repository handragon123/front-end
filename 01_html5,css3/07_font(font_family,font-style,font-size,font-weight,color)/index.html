홍익표 민주당 원내대표는 '김현정의 뉴스쇼'에서 "역대 대통령이 본인의 가족과 관련된 특검을 거부한 적은 없었다"면서 "김영삼 대통령은 검찰 수사를 허용한 거고 김대중 대통령도 수사를 받았고 특검을. 그 다음에 심지어 이명박 대통령도 본인의 가족과 관련된, 자신과 관련된 문제가 되니까 특검을 받았다"고 말했다.
